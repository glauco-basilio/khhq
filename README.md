# HalfQ Overlay Project

Projeto open-source para **Half‑QWERTY (mão esquerda)** usando **Karabiner‑Elements** + **Hammerspoon**.

## Instalação rápida
1. Descompacte este ZIP.
2. Rode `./scripts/install.sh` (pelo Terminal).
3. No Karabiner → Complex Modifications → **Add rule** → ative **HalfQ Left (US‑Intl) + Overlay**.
4. No Hammerspoon, **Reload Config**.

## Uso
- Segure **Space** → overlay exibe os pares (um quadrado: pressed em cima / sent embaixo).
- Solte **Space** → overlay some.
- Trainer de ritmo: **⌃⌥⌘ M** liga/desliga; **⌃⌥⌘ ↑/↓** ajusta BPM.
